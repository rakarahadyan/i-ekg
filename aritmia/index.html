Aritmia

Aritmia merupakan gangguan irama jantung yang terjadi akibat kelainan dalam pembentukan atau penghantaran impuls listrik di jantung. Dalam keadaan normal, impuls listrik berasal dari nodus sinoatrial (SA) yang berfungsi sebagai pacemaker alami. Impuls ini kemudian diteruskan melalui jalur konduksi ke atrium, nodus atrioventrikular (AV), dan ventrikel untuk menghasilkan kontraksi terkoordinasi. Bila terjadi gangguan pada mekanisme ini, baik karena pembentukan impuls yang abnormal atau hambatan konduksi, akan timbul aritmia (Nazmah, 2015; Surawicz & Knilans, 2019).

A. Aritmia karena Gangguan pembentukan impuls

a) Nodus SA

I. Takikardi sinus (ST)

• Irama : Teratur
• Frekuensi (HR) : > 100 - 150 kali/menit
• Gelombang P : Normal, setiap gelombang P diikuti gelombang QRS, dan T
• Interval PR : Normal
• Gelombang QRS : Normal

II. Bradikardi Sinus (SB)

• Irama : Teratur
• Frekuensi (HR) : < 60 kali/menit
• Gelombang P : Normal, setiap gelombang diikuti Gelombang QRS, T
• Interval PR : Normal
• Gelombang QRS : Normal

III. Aritmia Sinus

• Irama : Tidak teratur
• Frekuensi (HR) : Biasanya antara 60 - 100 kali/menit
• Gelombang P : Normal, setiap gelombang
• Interval PR : Normal
• Gelombang QRS : Normal
Semua gelombang sama

IV. Sinus Arest

• Irama : Teratur, kecuali pada yang hilang
• Frekuensi (HR) : Biasanya < 60 kali/menit
• Gelombang P : Normal, setiap gelombang P diikuti gelombang QRS
• Interval PR : Normal
• Gelombang QRS : Normal
• Hilangnya gelombang P, QRS dan gelombang T tidak menyebabkan jarak antara R–R

b) Atrium

I. Ekstrasistol atrial

• Irama : Tidak teratur karena ada irama yang timbul lebih awal
• Frekuensi (HR) : Tergantung irama dasarnya
• Gelombang P : Bentuknya berbeda dari irama dasarnya
• Interval PR : Normal atau memendek

II. Takikardi Supraventrikel

• Irama : Teratur
• Frekuensi (HR) : 150 - 250 kali/menit
• Gelombang P : Sukar dilihat, kadang terlihat tetapi kecil
• Interval PR : Tidak dapat dihitung atau memendek
• Gelombang QRS : Normal

III. Flutter atrial

• Irama : Biasanya teratur, bisa juga tidak teratur
• Frekuensi (HR) : Bervariasi, bisa normal, lambat atau cepat
• Gelombang P : Tidak normal, seperti gigi gergaji, teratur dan dapat dihitung (mis: P:QRS=2:1, 3:1 atau 4:1)
• Interval PR : Tidak dapat dihitung
• Gelombang QRS : Normal

IV. Fibrilasi atrial

• Irama : Tidak teratur
• Frekuensi (HR) : Bervariasi (normal, lambat atau cepat)
- Rapid Respon : HR ≥ 100 x/menit
- Normo respon : HR 60 - 100 x/menit
- Slow Respon : HR ≤ 60 x/menit
• Gelombang P : Tidak dapat diidentifikasi, sering terlihat keriting
• Interval PR : Tidak dapat dihitung
• Gelombang QRS : Normal

c) Nodus AV

I. Irama Junctional (JR)

• Irama : Teratur
• Frekuensi (HR) : 40 - 60 kali/menit
• Gelombang P : Terbalik di depan atau di belakang gelombang QRS
• Interval PR : < 0,12 detik atau tidak dapat dihitung
• Gelombang QRS : Normal

II. Ekstrasistol Junctional

• Irama : Tidak teratur, karena ada gelombang yang muncul lebih awal
• Frekuensi (HR) : Tergantung irama dasarnya
• Gelombang P : Tidak ada atau tidak normal, sesuai dengan letak impuls
• Interval PR : Tidak dapat dihitung atau memendek
• Gelombang QRS : Normal

III. Takikardi Junctional

• Irama : Teratur
• Frekuensi (HR) : > 100 kali/menit
• Gelombang P : Tidak ada atau terbalik di depan/belakang QRS
• Interval PR : Tidak dapat dihitung atau memendek
• Gelombang QRS : Normal

d) Supraventrikel

I. Ekstrasistol Supraventrikel (Ekstrasistol Atrial)

• Irama : Tidak teratur karena ada irama yang timbul lebih awal
• Frekuensi (HR) : Tergantung irama dasarnya
• Gelombang P : Bentuknya berbeda dari irama dasarnya
• Interval PR : Normal atau memendek

II. Takikardi Supraventrikel

• Irama : Teratur
• Frekuensi (HR) : 150 - 250 kali/menit
• Gelombang P : Sukar dilihat, kadang terlihat tetapi kecil
• Interval PR : Tidak dapat dihitung atau memendek
• Gelombang QRS : Normal

e) Ventrikel

I. Irama Idioventrikuler

• Irama : Teratur
• Frekuensi (HR) : 20 - 40 kali/menit
• Gelombang P : Tidak terlihat
• Interval PR : Tidak ada
• Gelombang QRS : Lebar (> 0,12 detik)

II. Ekstrasistol Ventrikuler

• Irama : Tidak teratur, karena ada gelombang yang timbul lebih awal
• Frekuensi (HR) : Tergantung irama dasarnya
• Gelombang P : Tidak ada
• Interval PR : Tidak ada
• Gelombang QRS : Lebar (> 0,12 detik)

III. Takikardi Ventrikel

• Irama : Teratur
• Frekuensi (HR) : > 100 kali/menit
• Gelombang P : Tidak terlihat
• Interval PR : Tidak ada
• Gelombang QRS : Lebar (> 0,12 detik)

IV. Fibrilasi Ventrikel

a. Fibrilasi Ventrikel (VF)
• Irama : Tidak teratur
• Frekuensi (HR) : Tidak dapat dihitung
• Gelombang P : Tidak ada
• Gelombang QRS : Tidak dapat dihitung, bergelombang dan tidak teratur

B. Aritmia karena Gangguan penghantaran impuls

a) Nodus SA

I. Blok sinoatrial (SA Block)

Terdapat episode hilangnya satu atau lebih gelombang P, QRS, T menyebabkan kelipatan jarak antara R-R
• Irama : Teratur, kecuali pada yang hilang
• Frekuensi (HR) : Biasanya < 60 kali/menit
• Gelombang P : Normal, selalu diikuti gelombang QRS
• Interval PR : Normal
• Gelombang QRS : Normal

b) Nodus AV

I. Blok AV derajat I (First degree AV block)

• Irama : Teratur
• Frekuensi (HR) : Biasanya antara 60 - 100 kali/menit
• Gelombang P : Normal, selalu diikuti gelombang QRS
• Interval PR : Memanjang > 0,20 detik
• Gelombang QRS : Normal

II. Blok AV derajat II Mobitz I (Wenckebach)

• Irama : Tidak teratur
• Frekuensi : 60 - 100 kali/menit atau < 60 kali/menit
• Gelombang P : Normal, tetapi ada satu gelombang P yang tidak diikuti Gelombang QRS
• Interval PR : Makin lama makin panjang sampai ada gelombang P yang tidak diikuti gelombang QRS, kemudian siklus makin panjang diulang
• Gelombang QRS : Normal

III. Blok AV derajat II Mobitz II

• Irama : Umumnya tidak teratur, kadang bisa teratur
• Frekuensi (HR) : Biasanya lambat, < 60 kali/menit
• Gelombang P : Normal, ada satu atau lebih gelombang P yang tidak diikuti gelombang QRS
• Interval PR : Normal / memanjang secara konstan kemudian ada blok
• Gelombang QRS : Normal

IV. Blok AV derajat III (Total AV Block)

• Irama : Teratur
• Frekuensi (HR) : < 60 kali/menit
• Gelombang P : Normal, tetapi P dan QRS berdiri sendiri-sendiri. Kadang diikuti QRS, kadang tidak
• Interval PR : Berubah-ubah
• Gelombang QRS : Normal atau > 0,12 detik

c) Interventrikuler

I. Right bundle branch block (RBBB)

• Irama : Teratur
• Frekuensi (HR) : Umumnya normal (60-100 x/menit)
• Gelombang P : Normal, setiap gelombang P selalu diikuti gelombang QRS dan T
• Interval PR : Normal
• Gelombang QRS : Lebar (> 0,12 detik)
• Catatan : Ada bentuk rsR’ (M shape) di V1 dan V2. Gel S yang lebar dan dalam di lead I, II, aVL, dan V6. Perubahan ST segmen dan gel T di V1 dan V2

• Irama : Teratur
• Frekuensi (HR) : Umumnya normal (60-100 x/menit)
• Gelombang P : Normal
• Interval PR : Normal
• Gelombang QRS : Lebar (> 0,12 detik)
• Catatan : Ada bentuk rsR’ (M shape) di V5 dan V6. Gel Q yang lebar dan dalam di V1 dan V2. ST segmen dan gel T di V5 dan V6.
